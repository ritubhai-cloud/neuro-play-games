# NeuroPlay Games

NeuroPlay is a single-page browser app for cognitive training games.

## Run Locally

Open `index.html` in a browser, or run:

```bash
npm start
```

## Deploy on Vercel

1. Push these files to `ritubhai-cloud/neuro-play-games`.
2. In Vercel, choose **Add New Project**.
3. Import the GitHub repository.
4. Leave the framework preset as **Other**.
5. Leave build command and output directory empty.
6. Deploy.

Vercel will serve `index.html` as the app entry point.
